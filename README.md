# **Lazur's Ores \[Darq] Texture Pack**



##### **About The Texture Pack**

**This texture pack is the 'dark' version of the** [**Lazur's Ores Texture Pack**](https://modrinth.com/resourcepack/lazurs-ores)**. This is a texture pack created by LazurMC that modifies the textures of the ores and deepslate ores, along with giving a dark overlay to it. In this context, the ores include the Coal Ore, the Iron Ore, the Copper Ore, the Lapis Lazuli Ore, the Redstone Ore, the Emerald Ore, the Diamond Ore, the Nether Quartz Ore, and the Nether Gold Ore, and the deepslate ores include all the mentioned ores' deepslate versions. This texture pack adds veins to the ore, along with adding a fresh new look to the ores, and making the Nether Quartz Ore's mineral much bigger (I have no idea why I did that). It also adds a few splash texts that give some fun bits of information about the texture pack and adds a dark overlay to all the ores' textures. The reason why we named it 'Darq' is because it is the Gen-Z version of the word 'Dark'! If you want to know more about how this texture pack was made or other details, head over to our official blog** [**here**](https://lazurmc.blogspot.com/) **and search for the relevant post. For instructions on how to use this texture pack, head over to our next section. Hope you enjoy using this texture pack! ##### **How To Use This Texture Pack?**

**First of all, shoutout to the** [**MCreator**](https://mcreator.net/) **development team for making the creation of this texture pack possible. Now, since this texture pack was created in MCreator 2025.3, the texture pack may appear incompatible since the latest version is 1.21.11 and the texture pack version is 1.21.8. But, don't worry, its actually not incompatible, since the versions are same, the game is by default supposed to inform you, but since this modifies the ores and adds a few texts, it would not be a problem. Follow the steps below to use this texture pack.**



1. **Install The Texture Pack: Head to one of the linked pages above, look for the corresponding texture pack, and download it. Make sure you don't extract the files from this ZIP file or make further modifications to this ZIP file except changing the name.**
2. **Locate The 'Resource Packs' Folder: Next, head to where the '.minecraft' folder is present. Open it, and locate the 'resourcepacks' folder. Your path may be this: "C:\\Users\\\[Your Username]\\AppData\\Roaming\\.minecraft\\resourcepacks". If you can't find one, don't worry! Just create a new folder and name it 'resourcepacks'.**
3. **Move The ZIP File To The Folder: Then, move the ZIP file to the 'resourcepacks' folder from your Downloads folder (or wherever you have downloaded). Ensure the folder name 'resourcepacks' is correct and that you have not extracted the ZIP file.**
4. **Enjoy Using Our Texture Pack: After you have completed these 3 steps, open your launcher, launch the game, and ensure that the texture pack is within the 'active resource packs' list. If not, move the texture pack to the list, and agree to the risk shown by the game saying that this texture pack is incompatible. After this, enjoy our special texture pack that modifies the ores and give it a new look. If you encounter any issues or bugs, go through all previous 3 steps, or report the bug by contacting us or commenting. Hope you enjoy the texture pack! :)**
